# PRO Рыбалка

Статический проект для браузера и Telegram Mini App.

## Запуск в браузере

Откройте папку через локальный HTTP-сервер, например:

```bash
python3 -m http.server 8080
```

Затем перейдите на `http://localhost:8080`.

## Telegram

1. Разместите файлы на HTTPS-хостинге.
2. Укажите URL Mini App через BotFather.
3. Для серверной авторизации проверяйте `Telegram.WebApp.initData` на backend. Не доверяйте `initDataUnsafe`.
4. `sendData()` применяется только для поддерживаемых способов запуска. Для полноценного сохранения прогресса подключите backend API.

## Файлы

- `index.html` — разметка и Telegram SDK.
- `style.css` — мобильный адаптивный интерфейс.
- `script.js` — механика, вероятности, эффекты и достижения.

## Звуковые файлы

Поместите рядом с `index.html` следующие OGG-файлы:

- `cast.ogg`
- `bonus.ogg`
- `debuff.ogg`
- `epic.ogg`
- `legendary.ogg`
- `achievement.ogg`
- `angus.ogg`
- `weather.ogg`

Публичная версия: одна игровая сессия в календарные сутки. Сессия сохраняется в `localStorage` и восстанавливается после перезагрузки страницы. Для конкурсного режима с защитой от очистки данных браузера требуется backend и проверка Telegram `initData`.

## Достижения

В игре реализовано 65 достижений: базовые, погодные, рыболовные, бонусные, артефактные, связанные с дебафами, Ангусом и редкими комбинациями событий. Полный список и точные условия доступны во встроенном справочнике.

## Визуальные эффекты

В этой версии добавлены CSS-анимации без внешних библиотек: заброс и всплеск, живой фон воды, погодные переходы, анимация новых записей, сцены тяжеловесов и гигантов, эпических и легендарных артефактов, Ангуса, дебафов и достижений. Кнопка ✨ в шапке переключает уменьшенный режим анимаций; настройка сохраняется в браузере.


## Баланс редкости

Базовые шансы: эпический артефакт — 1,2%, легендарный — 0,25%. Погодные режимы повышают эти веса, но артефакты остаются редкими. Условия наиболее частых достижений усилены и актуализированы в справочнике.


### Наглядность Касатки
Рыбы, съеденные Касаткой, остаются в хронологии с символом 💀 и затемнённым зачёркнутым оформлением. После нейтрализации Плавником мегалодона их обычный вид восстанавливается.


## Обновление хронологии
- Украденная Чайкой рыба помечается ❌ и затемняется до восстановления.
- Пояснения действий выводятся внутри карточки события.
- В строках живой рыбы периодически появляются силуэты под водой.
- Погодные сообщения окрашиваются в палитру режима.


Дополнительные звуки интерфейса в корне репозитория:

- `guide.ogg` — открытие справочника;
- `motion.ogg` — переключатель интенсивности анимаций.

## Мини-аркада

После начала сессии в свободных боковых зонах иногда проплывают малозаметные рыбы. Нажатие на силуэт добавляет рыбу в улов без расхода заброса. Обычные рыбы появляются чаще всего, тяжеловесы редко, гиганты исключительно редко. В хронологии такой улов отмечается значком 🐠. На него распространяются бонусы, дебафы и артефакты.


## Аркадная дорожка
Встроена в погодный блок и доступна на всех экранах. За сессию можно поймать до двух дополнительных рыб без расхода заброса.


## Аркадная дорожка v9

Аркадное событие может запуститься только после обычного заброса. Бездействие не вызывает появление рыбы. После каждого подходящего заброса шанс запуска составляет 8%, между событиями требуется минимум два обычных заброса. Рыба пересекает дорожку за случайные 2–4 секунды. За сессию можно поймать не более двух бонусных аркадных рыб. Счётчик появляется только на 4 секунды после успешной ловли. С вероятностью 5% вместо рыбы появляется более крупный силуэт Касатки; нажатие активирует дебаф Касатка и не учитывается в лимите бонусного улова.


## Изменения v10
- Статистика забросов и текущего веса отображается компактным текстом справа внутри погодного блока.
- Погодный блок уменьшен и закрепляется при прокрутке.
- Внутренняя прокрутка хронологии удалена.
- Касатка больше не создаёт отдельные строки при поедании рыбы: состояние отображается прямо в строке рыбы и отменяется при восстановлении.

## Обновление v11
- Игровая сцена зафиксирована в пределах экрана; прокручивается только хронология, полоса прокрутки скрыта.
- Глубоководное нечто преобразует хлам внутри исходной строки: зачёркнутый хлам → полученная рыба.
- Последующие действия Чайки, Касатки и восстановлений отображаются в той же строке трансмутации.
- Тестовая сборка не ограничивает количество сессий.

## Уточнение баланса Акваланга

Акваланг применяется в финале один раз к самой тяжёлой оставшейся рыбе. Несколько экземпляров складываются линейно: 1 — ×3, 2 — ×6, 3 — ×9. Штурвал Наутилуса удваивает силу каждого Акваланга: ×6, ×12 и ×18 соответственно.
