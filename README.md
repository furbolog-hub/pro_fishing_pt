# Игровой портал GitHub Pages

## Структура

- `index.html` — корневое меню выбора игр.
- `menu.css` — оформление меню.
- `.nojekyll` — должен оставаться только в корне репозитория.
- `dropfish/` — полностью автономная игра DropFish.

## Звуковые файлы PRO Рыбалки

Положите звуки в папку `dropfish/` рядом с `script.js`:

- `cast.ogg`
- `bonus.ogg`
- `debuff.ogg`
- `epic.ogg`
- `legendary.ogg`
- `achievement.ogg`
- `angus.ogg`
- `weather.ogg`
- `guide.ogg`
- `motion.ogg`

Пути в коде уже относительные (`./cast.ogg`, `./guide.ogg` и т. д.), поэтому при переносе всей игры в папку переписывать их не требуется.

## Публикация

Загрузите содержимое архива в корень репозитория GitHub Pages. Главная ссылка откроет меню, а игра будет доступна по адресу с папкой `dropfish`.

Для Telegram Mini App укажите URL корневой страницы репозитория. Игрок сначала увидит меню выбора игр.
